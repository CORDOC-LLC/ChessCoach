# Suggested App Store / Website Screenshots

Four screenshots, same set reused for both the App Store Connect listing and
the website's `ScreenshotFrame` placeholder slots
(`chesscoach-website/public/screenshots/`).

## 1. Hero / hook

- **Caption:** "Stop making the same mistakes."
- **Subcaption:** "A real chess engine grades every move — and explains why."
- **Capture:** Play mode mid-game with a clear coach explanation visible.
  This is the highest-stakes shot — App Store search results often show only
  screenshot 1.
- **Website file:** `hero-play-mode.png`

## 2. Best-move hints + opening ID

- **Caption:** "See the best move — and the opening you're playing."
- **Capture:** Live game with hint arrows active (best + alternative move)
  and the opening name label visible (e.g. "Queen's Pawn Game · A40").
- **Website file:** `feature-hints.png`

## 3. Coach in action

- **Caption:** "A coach that explains why, not just what."
- **Capture:** The Coach card with a real written explanation showing,
  ideally mid-conversation in the chat view so it reads as a live dialogue,
  not a static tip.
- **Website file:** `feature-coaching.png`

## 4. Board scanning

- **Caption:** "Snap a photo. Get instant analysis."
- **Capture:** The board scanner mid-scan or just after recognizing a
  physical board, transitioning into analysis.
- **Website file:** `feature-scan.png`

## Notes

- Headline for screenshot 1 was chosen over "Like a coach watching your
  shoulder" to match the website hero, paywall, and onboarding voice, and
  because a specific pain point stops the scroll faster than an atmospheric
  description.
- Captions are sized for the bold overlay text App Store screenshots
  typically use — adjust wording once laid over real device frames.
- Puzzles (`feature-puzzles.png`) didn't make the 4 App Store slots but still
  has its own website placeholder slot if a 5th screenshot gets added later.
